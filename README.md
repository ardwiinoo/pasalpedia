# pasalpedia

### Clone Repo

```bash
    git clone git@github.com:ardwiinoo/pasalpedia.git
```

### Enter Directory

```bash
    cd ./pasalpedia
```

## Install Deps

-   python
-   nodejs

```bash
    pip install -r requirements.txt
```

```bash
    npm install
```

## Run Project

Terminal 1

```bash
    npm run buildcss
```

Terminal 2

```bash
    python main.py
```
